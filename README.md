# RecruitmentSite_project
학과 채용 관련 홈페이지

#기능
회사추가 버튼을 누르면, 회사 추가 가능
삭제 버튼을 클릭하면, 삭제 가능 
#적용 기술
json server 및 REST API를 이용하여, 데이터를 처리
React 아래 기술 이용
컴포넌트를 구성하여 구현
라우터 이용
리액트 훅스 이용
React-Boot  이용 UI/UX 개선
